API

Read-only REST access to the same records powering this console. Bearer-token auth.

v1 · stable
rate limit 600 req / 5 min
Base URL
https://api.planmytransfer.org/v1
Endpoints
GET/agreementsList agreements by campus, program, college
GET/agreements/{id}One agreement with full requirement ledger
GET/courses/sendingSearch the community-college catalog
GET/courses/receivingSearch the university catalog
GET/analysis/coverageAggregate coverage statistics
Example request
curl -s https://api.planmytransfer.org/v1/agreements \
  -H "Authorization: Bearer $PMT_TOKEN" \
  -G --data-urlencode "receiving_campus=UC San Diego" \
     --data-urlencode "program=B.S. Computer Science" \
     --data-urlencode "dataset=v2025.2"
