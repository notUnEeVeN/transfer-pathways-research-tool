Prerequisite graph — swim-lane redesign

Replaces the hand-tuned absolute-position mockup. Rows are disciplines, columns are prerequisite depth, so every chain reads left-to-right with no crossings inside a lane.

Why lanes
Chains own their row

The calc spine, both physics sequences, and CS1→CS3 each claim a full row, so the backbone of every discipline is a straight line — the old layout bent these around unrelated nodes.

Why depth columns
Position encodes "how deep"

A course's column is its longest prerequisite path. Researchers can answer "how many semesters of runway does this require?" by reading the header — no tracing needed.

Cross-lane edges
Bundled, routed in gutters

Edges leaving one course (Calc I feeds five things) merge into a single trunk that branches per target, colored by source discipline. Parallel trunks get separate gutter tracks instead of overlapping.

Focus mode
Click to trace a chain

Clicking a node fades everything outside its ancestor/descendant closure and reports upstream/downstream counts. Esc clears. The rules table below the graph stays as the zero-interaction fallback.
